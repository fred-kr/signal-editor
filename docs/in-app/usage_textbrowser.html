Requirements

European Data Format (.edf)

  • Has at least 3 channels with `ch0` containing temperature, `ch1` containing heartbeat and `ch2` containing ventilation data
  • The name of the temperature channel contains the sub-string "temp", the name of the heartbeat channel contains the sub-string "hb" or "heart" and the name of the ventilation channel contains the sub-string "vent" (all case-insensitive)

Feather (.feather)

Required column structure:

│ index ┆ time_s ┆ temperature ┆ hbr ┆ ventilation │
│ --- ┆ --- ┆ --- ┆ --- ┆ --- │
│ u32 ┆ f64 ┆ f64 ┆ f64 ┆ f64 │

A more general approach will (hopefully) be implemented sometime in the future.

Data selection

  1. Click Select File and choose a compatible data file to load
  2. The active working data can be limited by selecting a column name from the dropdown menu along with specifying the upper and lower allowed limits. The lower limit is inclusive, the upper exclusive (setting the lower limit to 10 and the upper to 100 would result in the values 10 - 99 being loaded)
  3. To work with the entire dataset, just leave the Specify subset checkbox unchecked
  4. Finally, use the Load button to load the selected range of data into the program.
